# Documentation

This folder contains development and contributor documentation for the Live Blog Writer extension.

## Files

- **CONTRIBUTING.md** - Guidelines for contributing to the project
- **DEVELOPMENT.md** - Development setup and workflow
- **TESTING.md** - Testing procedures and guidelines
- **DEMO.md** - Demo script and usage examples
- **PROJECT_SUMMARY.md** - High-level project overview
- **SECURITY_SUMMARY.md** - Security considerations and practices
- **ICON.md** - Icon design specifications and requirements
- **FINAL_STATUS.txt** - Project completion status

## User Documentation

User-facing documentation remains in the root directory:
- **README.md** - Main project documentation (shown on GitHub and marketplace)
- **QUICKSTART.md** - Quick start guide for users
- **CHANGELOG.md** - Version history and changes
- **LICENSE** - License information
